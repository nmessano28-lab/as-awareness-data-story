# Portfolio Case Study: When Back Pain Is More Than Back Pain

## Project Type

Interactive public awareness data story

## Project Summary

I created a web-based data story about ankylosing spondylitis, a chronic inflammatory condition that can be difficult to recognize because it often begins as back, neck, or hip pain. The project combines personal experience, public health sources, structured data, and interactive visuals to make an invisible chronic illness easier to understand.

## Why I Built It

As someone living with ankylosing spondylitis, I wanted to use data visualization to explain a condition that is often invisible from the outside but deeply present in daily life. My goal was to create something personal, educational, and professionally useful: a project that shows data storytelling skill while helping people understand why awareness and rheumatology access matter.

## Data & Sources

The project uses public source material from:

- National Institute of Arthritis and Musculoskeletal and Skin Diseases
- Spondylitis Association of America
- Verywell Health diagnostic-delay summary
- PubMed-listed disease-status measurement reference

I converted source-backed concepts into small structured datasets for:

- Key AS awareness facts
- Age-of-onset markers
- Diagnosis journey stages
- Symptom and body-system impact areas
- Visible vs. invisible illness comparisons

## Skills Demonstrated

- Research synthesis
- Data storytelling
- Health communication
- Structured data creation
- Interactive visualization
- HTML, CSS, and JavaScript
- Source documentation and caveat writing
- Responsive web design

## Key Design Choices

- Used a personal opening note to establish why the topic matters.
- Placed the medical disclaimer near the beginning to keep the project responsible.
- Used plain-language explanations instead of clinical jargon.
- Built interactive visuals for onset, diagnosis journey, symptom burden, and invisible illness.
- Included source cards so viewers can trace the public information behind the story.

## Portfolio Description

**When Back Pain Is More Than Back Pain**  
Built a web-based public awareness data story about ankylosing spondylitis using public health sources, structured data, and interactive visual design. The project explains why AS can be misunderstood, how diagnosis can be delayed, and how invisible symptoms can affect daily life. It combines personal narrative with health communication and data visualization.
