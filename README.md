# When Back Pain Is More Than Back Pain

A publishable static website and portfolio data-story project about ankylosing spondylitis awareness.

## Project Angle

**When Back Pain Is More Than Back Pain** is a public awareness data story about ankylosing spondylitis. It combines personal narrative, public health research, structured data, and interactive visual design to explain why AS can be misunderstood, delayed, and invisible from the outside.

## What This Project Shows

- Public-source research synthesis
- Health communication
- Data storytelling
- Interactive web design
- Structured data creation
- Accessibility-conscious source notes and medical disclaimers

## Files

- `index.html` - main webpage
- `style.css` - visual design and responsive layout
- `script.js` - interactive charts and generated content
- `data/as_story_data.js` - structured source-backed story data
- `docs/portfolio_case_study.md` - case-study summary for portfolio use
